Get or create a remote + cloned repository.

Parameters

  • details (RepositoryDetails) – A description of the repository full URL, the group it’s nested under, and any other information that’s important for cloning.

  • destination (str) – The absolute path to a directory on-disk to clone into. This path should be empty.

  • auto_create (bool, optional) – If the git repository doesn’t already exist and auto_create is True, make it and return the newly created repository.

Raises

  • RuntimeError – If destination is not empty.

  • RemoteActionFailed – If details could not be cloned.

get_all_publishers(package)[source]

Get every publish method registered globally and under package.

Important

The returned Publisher classes are not authenticated by default. You need to call Publisher.authenticate() to connect them to a remote repository database (like GitHub).

Parameters

package (rez.packages.Package) – A package which may contain extra configuration overrides. Any override not found will be retrieved globally.

Returns

Every registered publish method.

Return type

list[Publisher]

get_first_versioned_view_url(package, allow_optionals=False)[source]

Find a URL where published documentation lives.

Parameters

  • package (rez.packages.Package, optional) – A package which may contain extra configuration overrides. Any override not found will be retrieved globally.

  • allow_optionals (bool, optional) – In rez_docbot’s configuration, users can mark documentation sources as optional (or not). If it’s optional, there’s a chance that the documentation URL points to nothing. If you would rather ensure the returned URL definitely has documentation, use the default option for this parameter (False).

Raises

RuntimeError – If no viewing URL could be found.

Returns

A recommended, found URL to view the documentation from.

Return type

str

class Handler[source]

Bases: object

An adapter class to make working with remote git services easier.

This class helps initialize BaseRepository objects, which is where all of the “real” remote calls are going to be sent during publishing.

abstract get_repository(details, destination, auto_create=True)[source]

abstract get_resolved_view_url()[source]

Create a viewable URL where documentation can be seen.

This differs from the publish URL. For example, the publish URL for GitHub is typically "git@github.com:UserName/{package.name}" but the documentation is viewed in "https://UserName.github.io/{package.name}".
